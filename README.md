# Moses Bones Bot

All-in-one Discord bot: friendly AI chat, moderation, welcome messages and daily jokes.

## Commands
!help  – show help  
!joke  – random joke  
!warn @user reason – warn (staff)  
!warns @user – check warns  
!kick @user reason – kick (staff)  
!ban @user reason – ban (staff)

Bot only works in the configured channel (general chat).
